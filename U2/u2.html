Grunder

HTML

Meta

html - huvudelement som innehåller hela htmläfiles element

Head - beskriver dokumentet till datorn

Style - Hanterar stilen på alla element på hemsidan

Layout

header - sidhuvud innehåller information om sidan. Kan ev innehålla rubrik

main - innehållet på sidan innehåller flera sections

title - definerar titlen på hemsidan, dels visar den titeln på fliken

footer - Är botten på hemsidan och har alla element som är där

Text och rubriker

h1,h2,h3,h4,h5,h6 - Olika headers, det hjälper med organisering

p - paragrafer som man kan skriva text i

CSS

Typsnitt och stilar

font-family - bestämmer vilket typsnitt som ska användas

font-size - anger storlke

font-style - gör texten till kursiv eller normal

word-spacing - anger avståndet mellan orden

färger

color - ändrar färgen på texten

background-color - sätter bakgrundsfärgen på ett element

background - anger samma men är mer kraftfull (bilder, position osv)

layout

display - Utgör hur element visas

top, right, bottom, left - position för element som placeras

widht / height - elements bred och höjd

margin - yttre avstånd runt element

padding - inre avstånd mellan innehåll

border - utgör tjocklek, färg eller stil på en kantlinje

display: flex - aktiverar flex layout

align-items - vertikal justering av ett element

gap - avstånd mellan rader och kolumner
